# Thriveon – Cloud-Based Task Management System

**Thriveon is a full-stack web application designed to help users manage and track their daily tasks efficiently. Built using the MERN stack and deployed on cloud platforms, it demonstrates both frontend and backend integration using modern tools.**

- 🚀 Frontend Repository: [Thriveon Frontend](https://github.com/astel-code/thriveon_frontend)  
- ⚙️ Backend Repository: [Thriveon Backend](https://github.com/astel-code/thriveon_backend)

## 🛠️ Tech Stack

- Frontend: React.js, HTML, CSS  
- Backend: Node.js, Express.js  
- Database: MongoDB (MongoDB Atlas)  
- Deployment: Netlify (Frontend), Railway (Backend)  

## ✅ Key Features

- User authentication and secure session management  
- Task creation, editing, deletion, and completion tracking  
- RESTful APIs for task operations  
- Middleware integration for backend validation  
- Responsive UI and cloud-hosted infrastructure
- Priority-based task management with visual matrix
- Hierarchical task organization
- Interactive Gantt chart for project timeline management
- Digital sticky notes for quick reminders
- Multiple view modes (Dashboard, Tasks, Priority Matrix, etc.)


## 🎯 Application Features Showcase

### 📊 **Task Management Views**
- **Dashboard**: Quick overview of daily, weekly, and monthly tasks
- **Priority Matrix**: Categorize tasks by urgency and importance
- **Hierarchical List**: Organize complex projects with subtasks
- **Gantt Chart**: Visual timeline for project planning
- **Notes**: Digital sticky wall for quick reminders

### 🎨 **User Interface**
- Clean, modern dark theme design
- Intuitive navigation with sidebar menu
- Responsive layout for all screen sizes
- Interactive elements with hover effects
- Progress bars and completion indicators

### 📱 **Navigation Features**
- Dashboard - Main overview
- Tasks - Detailed task management
- Priority Matrix - Eisenhower matrix view
- Hierarchical List - Nested task organization
- Gantt Chart - Timeline visualization
- Notes - Sticky notes functionality
- MindMap - Visual task mapping
- Blogging - Task-related documentation
- Word(Docx) - Document integration

1. Clone the frontend and backend repositories
2. Install dependencies in both projects
3. Set up MongoDB Atlas connection
4. Configure environment variables
5. Run the development servers
6. Access the application at your local development URL
